# FFCS On The Go — Vellore Campus

A fully functional FFCS timetable planner for VIT Vellore students.

## Features
- Complete VIT Vellore FFCS slot grid (MON–FRI, Theory + Lab slots)
- Add courses with Code, Title, Faculty, Venue, Credits
- Slot conflict detection
- Quick Visualization mode
- Download as PDF (Print) or CSV
- Reset timetable
- Auto-saves to browser localStorage
- Double-click any cell or course row to reload it into the panel

## Files
```
index.html      — Main page structure
style.css       — All styles
timetable.js    — All logic and data
```

## Local Usage
Just open `index.html` in any browser. No server needed.
